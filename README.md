¡Qué tal, amigos y apasionados por la ciberseguridad! Durante mis estudios para la certificación CompTIA CySA+, noté que había muy poco material de práctica en español. Por eso, dediqué un tiempo a traducir una guía de SYBEX con 1000 preguntas que cubren todos los dominios del examen.

Es importante mencionar que estas preguntas no son de mi autoría; mi aporte consistió únicamente en traducirlas para nuestra comunidad. Desarrollé este simulador utilizando Python y JSON. Aunque quizás le haga algunos ajustes en el futuro, estoy seguro de que les será de gran ayuda para aprobar. El simulador incluye imágenes y logs reales para que practiquen el análisis como todos unos expertos.
